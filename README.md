# projeto-ransomware-python
Projeto desenvolvido com fins educacionais, demonstrando de forma controlada o processo de criptografia e descriptografia de arquivos em Python, simulando o funcionamento de um ransomware para estudo em Segurança da Informação.

O repositório contém o código-fonte, documentação detalhada no README.md e arquivos de teste utilizados exclusivamente em ambiente de laboratório.

Os prints dos testes de execução (antes da criptografia, após a criptografia e após a descriptografia) estão disponíveis no link abaixo:
